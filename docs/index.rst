Welcome to SMILES-TO-IUPAC-name translator web's documentation!
===============================================================
.. image:: https://github.com/Kohulan/cheminf-jena-logos/blob/main/STOUT/STOUT.png?raw=true
  :width: 500
  :align: center

STOUT-V2, SMILES to IUPAC name translator using transformers. STOUT-V2 can translate SMILES to IUPAC names and IUPAC names back to a valid SMILES string. STOUT is actively developed and maintained by Kohulan Rajan at the Steinbeck group at the Friedrich Schiller University Jena.

How to access
=============

https://stout.decimer.ai
